# 腾讯 Java 语言编程指南

《腾讯 Java 语言编程指南》是腾讯代码委员会 Java 分会基于 Code Review 总结并在公司范围广泛征集意见得出的编程实践集合。本指南从 Java （JDK 8以后版本）的语言特性出发，结合日常工作中碰到的实际问题，展示了在 Java 编程中常见的问题和**推荐**方案。希望本手册能够帮助你写出更易读、更健壮、性能更好的 Java 代码。
